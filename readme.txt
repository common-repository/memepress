== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `the directory "memepress"` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Memepress is added to "Options" in the WordPress dashboard.
